# Sonic Soulitons Restaurant Sever

## Requirement

Node version: 18+

## TL;DR

```sh
yarn install
```


## CommitRule

Follow the commitlint pattern: subject: message

subject would be one of the following below:

- build
- chore
- ci
- docs
- feat
- fix
- perf
- refactor
- revert
- style
- test

Real world example
```sh
feat: config core module
```
